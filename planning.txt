Order Book Simulator
A simple simulation of a trading system that deals with limit orders
1. Add limit order (buy/sell, price, quantity)
2. Delete order 
3. Match trades whenever best bid >= best ask 
4. Maintain best bid/ask and print trades + top of book 

*Define input format: 
- ADD: A, id, (buy/sell), price, quantity
- DELETE: D, id 
- Print trades as they happen 

- Trade at the resting order price: 
When new order is added, treat like an incoming order and match
with the opposite side
